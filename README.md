\# ArgusQ Dashboard



Streamlit dashboard for the ArgusQ semiconductor process monitoring project.



\## App entrypoint



pipeline/dashboard/app.py



\## Deployment



This dashboard is intended for Streamlit Community Cloud demo mode.



Live mode requires the full local ArgusQ stack:

\- Kafka

\- Spark Structured Streaming

\- MinIO / Delta Lake

\- Airflow



For public Streamlit deployment, use Demo mode.
